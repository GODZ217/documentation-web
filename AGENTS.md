## Node.js

This project requires Node.js >= 20. Use nvm to manage the Node version:

```bash
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
nvm use 20
# or nvm use default (which points to v20.20.2)
```
